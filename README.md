# arduino-TVout
Library for PAL signal communication to analog tv using arduino UNO.
Schematics and PCB files included.

ARDUINO UNO 
13-------SPI CLK

11-------SPI SI

10-------SYN

7--------CS_0

6--------E_CLK

5--------CS_1

4--------CS_2

1--------MUX_0

0--------MUX_1

if it doesn't work try to swap mux_0 and mux_1.
